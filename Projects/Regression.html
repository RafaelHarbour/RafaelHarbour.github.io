import statsmodels.api as sm
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error
X_linear = sm.add_constant(X)
X_train, X_test, y_train, y_test = train_test_split(X_linear, y, test_size = 0.2, random_state = 19)
mod = sm.OLS(y_train, X_train)
res = mod.fit()

We use train_test_split() to split our data for our model. 80% of the data will be used for training and then we will test our model with 20% unseen test data. We use a fixed random state across both the MLR and random forest model to keep the training and testing data the same so we can then fairly compare the models.

There are some interesting statistics in the OLS regression results above: The R-squared value is a measure of how good our model is to our data. A score of 0.989 signifies that 98.9% of the data can be explained by our model. This demonstrates that this is a good model for our data. The p values ("P > |t|" in the chart) show the significance of the null hypothesis. A low p value means that we should reject the null hypothesis (the null hypothesis is that the coefficient of the respective variable is 0 and therefore should not be in our model), in this case all p values are 0.000 and therefore there is significant evidence that all variables are necessary in our model. We can write our model in the form of y = bx + c where c is the constant, b is the coefficient row vector, x is the independent variables column vector and y is dependent vector. The equation of our model is y = 36.91 + 0.25Solid Fuels - 0.07Gas + 0.12Electricity - 0.16Liquid Fuels + 0.41Motor fuel and oil (2dp)

The RMSE for the random forest model is less than the MLR model, therefore we can conclude that in this scenario the random forest model is better. The benefit of the MLR model is that we can write the model down as an equation and see the weights (coefficients) of each independent variable. The benefit of the random forest regression model is that it looks deeper into patterns and relationships between the variables. It's a strong model out of the box but there is no equation to see how the independent variables are used in calculating the dependent variables.
